htbaapub.fuzzy

Fuzzy Logic

BlitzMax translation of Mat Bucklands' C++ Fuzzy Logic implementation described in the book Programming Game AI by Example.

What use is it?

With Fuzzy Logic it's possible to let your game agents make decisions like humans do. A nice tutorial/explanation can be found at the Dream Makers website.

Examples

Please take a look at one of the examples below.
